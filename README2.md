# Крестики-нолики

Простая реализация игры «Крестики-нолики» на HTML, CSS и чистом JavaScript.

## Описание игры

### Правила игры
- Игроки по очереди ставят свой символ (`X` или `O`) в свободные клетки поля 3×3.
- Первый ход делает игрок за `X`.
- Цель — первым выстроить три своих символа в ряд: по горизонтали, вертикали или диагонали.
- Если все 9 клеток заполнены, и никто не выстроил ряд — объявляется ничья.

### Логика победы/поражения
- Победа определяется сразу после хода, если один из игроков собрал выигрышную комбинацию.
- После победы игра блокируется — дальнейшие клики не обрабатываются.
- При ничьей игра также завершается, и победитель не объявляется.

### Особенности реализации
- Игра полностью работает в браузере без использования внешних библиотек.
- Интерфейс адаптирован под разные устройства за счёт `grid` и гибких размеров.
- Есть кнопка перезапуска игры без перезагрузки страницы.

## Игровая механика

### Как определяется победитель?
Функция `checkResult()` проходит по массиву предопределённых выигрышных комбинаций `winConditions`. Если находит три подряд одинаковых символа (`X` или `O`), игра завершается победой текущего игрока.

### Как обрабатываются клики?
Функция `handleCellClick`:
- получает индекс клетки из атрибута `data-index`;
- проверяет, свободна ли клетка и активна ли игра;
- если всё в порядке — обновляет массив `board`, отображает символ в клетке;
- вызывает `checkResult()` для проверки победы или ничьей.

### Как работает перезапуск?
Функция `restartGame()`:
- сбрасывает массив `board` в исходное состояние (пустые строки);
- устанавливает `currentPlayer = "X"`;
- устанавливает `gameActive = true`;
- очищает текстовое содержимое всех клеток;
- обновляет строку статуса.
